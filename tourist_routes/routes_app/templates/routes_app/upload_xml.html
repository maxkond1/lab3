Загрузка XML файла с маршрутами

{% csrf_token %}

Требования к XML файлу:

Загрузка будет ОТКЛОНЕНА если:

  • Отсутствует любой из обязательных полей
  • Обязательные поля пустые
  • Числовые поля содержат нечисловые значения
  • Числовые поля содержат отрицательные значения

Обязательные поля (должны быть в каждом маршруте):

  • <name> - Название маршрута
  • <description> - Описание
  • <length_km> - Протяженность в км (положительное число)
  • <duration_days> - Продолжительность в днях (целое положительное число)
  • <difficulty> - Сложность (легкий/средний/сложный)
  • <region> - Регион

Пример корректного XML:

<?xml version='1.0' encoding='utf-8'?>
<tourist_routes>
    <route>
        <name>Поход на Эльбрус</name>
        <description>Восхождение на высочайшую вершину России</description>
        <length_km>42.5</length_km>
        <duration_days>7</duration_days>
        <difficulty>сложный</difficulty>
        <region>Кабардино-Балкария</region>
        <best_season>июль-август</best_season>
        <kolvo_chel>10</kolvo_chel>
    </route>
</tourist_routes>
